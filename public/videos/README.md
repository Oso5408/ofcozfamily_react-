# Videos Directory

Place your video files here.

## Required File
- `hero-video.mp4` - Main hero section video

## Optional
- `video-poster.jpg` - Video thumbnail/poster image (place in `/public/`)

## Instructions

1. Download your video file
2. Rename it to `hero-video.mp4`
3. Copy it to this directory: `public/videos/hero-video.mp4`
4. Refresh your browser

The video will automatically load on the homepage.

## Recommended Video Specs
- Format: MP4 (H.264)
- Resolution: 1920x1080 or 1280x720
- File size: < 20MB
- Bitrate: 2-5 Mbps
